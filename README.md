# flask-chatbot
Built on python 3.6
Flask==0.11
chatterbot==0.8.4
SQLAlchemy==1.1.11

#### A web implementation of [ChatterBot](https://github.com/gunthercox/ChatterBot) using Flask.

## Local Setup:
 1. Open command prompt and locate folder. run 'pip install -r requirements.txt'
 2. Run *train.py*
 3. Run *run.py*
 4. Demo will be live at http://localhost:5000/

 ## Usage: 
 Banking chatbots simplify customers' daily lives by providing instant assistance 24/7. From checking account balances to making transfers, these bots handle routine tasks efficiently. They offer personalized solutions, answer queries, and guide users through complex processes, enhancing convenience and accessibility.
